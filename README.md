# Noufanudheen — Personal Profile Page

A premium, animated personal portfolio page built with **pure HTML & CSS** — zero JavaScript, zero frameworks.

---

## ✨ Design Language

| Layer | Style |
|---|---|
| Cards & Buttons | **Glassmorphism** — frosted glass with backdrop blur |
| Stat & Secondary cards | **Neumorphism** — soft dark extrusion/inset shadows |
| Background | Animated CSS polygon shapes, ambient orbs, dot grid |
| Hero | Spinning CSS rings with glow dots, SVG polygon frame |

## 🎨 Theme

- **Background** — Deep black `#050508`
- **Primary accent** — Electric blue `#00aaff`
- **Secondary accent** — Red `#ff2255`
- **Typography** — White-ish `#f0f4ff`
- **Fonts** — [Space Grotesk](https://fonts.google.com/specimen/Space+Grotesk) + [Outfit](https://fonts.google.com/specimen/Outfit)
- **Icons** — [Google Material Symbols Outlined](https://fonts.google.com/icons)

## 🗂 Sections

- **Hero** — Name, CSS typing animation, CTA buttons, social links
- **About** — Bio card (glass) + stat cards (neumorphic)
- **Tech Stack** — Frontend · Backend · Tools with animated progress bars
- **Projects** — Cards pulling from pinned GitHub repos
- **Contact** — Email · GitHub · LinkedIn cards + CTA button

## ⚙️ Tech

- HTML5 (semantic — `header`, `main`, `section`, `article`, `footer`)
- Vanilla CSS3
  - CSS custom properties (design tokens)
  - `@keyframes` animations (typing, skill bars, entrance, float, spin)
  - `clip-path` polygon shapes
  - `backdrop-filter` glassmorphism
  - CSS checkbox hack for mobile nav (no JS)
- Google Fonts (Outfit, Space Grotesk, Material Symbols Outlined)
